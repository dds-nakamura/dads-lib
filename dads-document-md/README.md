## 取得手法

`/dads/` 配下の全公開ページを並列エージェントで取得し、URLパスと一致するフォルダ構造で `dads/<path>/index.md` として保存しています。本文は原文（日本語）を維持しつつ、Markdownの見出し・表・コード・リストに整形しています。

## 注意事項

- 一部のコンポーネントは取得時点でガイドラインが「準備中」と表記されていました。
- スクロールトップボタン（Scroll Top Button）はサイト上で **deprecated** と明記されています。
- いくつかの `/usage/` `/accessibility/` サブページは取得時にHTTP 403を返したため、可能なものは個別に再取得しています。
- Figma、GitHub、外部リソースは元サイトの該当リンクを参照してください。
