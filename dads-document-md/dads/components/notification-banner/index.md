# ノティフィケーションバナー（概要）

> **ソース**: https://design.digital.go.jp/dads/components/notification-banner/
> **スクレイプ日**: 2026-05-12
> **説明**: デジタル庁デザインシステムβ版 - ノティフィケーションバナーコンポーネント概要
> **最終更新**: 2025年1月9日

## 概要

サイト/サービス全体に関わる、またはページや要素単位における重要度の高い情報を、ユーザーの操作に関わらず、サイト/サービス側からユーザーへ提示する場合に用いる通知バナーです。

## バリアント（通知タイプ）

| タイプ | 用途 |
|------|------|
| サクセス（Success） | 操作等の滑りない完了の通知 |
| エラー（Error） | 操作等が正しく完了しなかったことの通知 |
| 警告（Warning） | 注意を喚起するための警告、警戒情報等の通知 |
| 情報提示 Info 1 | プライマリカラー（青）を使用した中立的な情報提示 |
| 情報提示 Info 2 | ニュートラルブラックを使用した一般的な情報表示 |

## デザインスタイル

| スタイル | 特徴 |
|---------|------|
| スタンダードスタイル | 角丸ボーダーデザイン |
| カラーチップスタイル | 左側に目立つボーダーアクセント |

## パーツ

### 必須要素

- 通知アイコン
- バナータイトル

> **注記**: バナータイトルはノティフィケーションバナーの必須構成要素です。

### 任意要素

- 閉じるボタン
- 日時テキスト
- バナー説明文
- アクションボタン

## 使い方

### 配置

ノティフィケーションバナーはページの上部のファーストビューで必ずユーザーに見える箇所へ配置し、視認性を高めます。

| 配置場所 | 詳細 |
|---------|------|
| ページ上部（推奨） | 視認性を最大化するためページ上部に配置 |
| コンテンツ内特定セクション | 関連するセクション内に配置 |
| 固定配置 | 閉じるボタンの設置が必須 |

### 閉じるボタン

閉じるボタンはバナーを非表示にし、その状態をストレージに保存します。

### アクションボタン

デバイスのビューポートを考慮したボタン数の制限があります。

## アクセシビリティ

| 要素 | コントラスト比要件 |
|------|----------------|
| テキスト要素 | 隣接する背景に対して最低 4.5:1 |
| 非テキスト要素 | 隣接する背景に対して最低 3:1 |
