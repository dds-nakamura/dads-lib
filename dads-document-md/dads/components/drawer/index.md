# ドロワー（概要）

> **ソース**: https://design.digital.go.jp/dads/components/drawer/
> **スクレイプ日**: 2026-05-12
> **説明**: デジタル庁デザインシステムβ版 - ドロワーコンポーネント概要
> **最終更新**: 2025年1月15日

## 概要

ブラウザ画面の四辺（上下左右端）から展開し、モバイルメニューなどのコンポーネントを格納可能なコンテナです。

## 仕様

### 基本動作

- ドロワーは常にモーダルとして動作します。
- 親要素はオーバーレイシェードで覆われるエレベーション設定が適用されます。
- ドロワーのコンテンツとリンク以外のコンテンツへのアクセスはブロックされます。
- 専用の閉じるボタンによりドロワーを閉じることができます。

### 展開パターン

| パターン | 説明 |
|---------|------|
| フルオーバーレイ | 画面全体を覆うように展開 |
| 右側オーバーレイ | 画面右端から展開 |
| 左側オーバーレイ | 画面左端から展開 |
