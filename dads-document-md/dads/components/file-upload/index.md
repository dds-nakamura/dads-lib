## 概要

ファイルアップロード／ドロップエリアは、HTMLの `<input type="file">` に対応した、単体または複数のファイルを選択してアップロードできるようにするためのコンポーネントです。ドラッグ＆ドロップでのファイル選択にも対応しています。

---

## ユースケース

- フォームでファイルのアップロードを受け付けるとき
- ウェブアプリケーションにおけるファイルの一覧画面で、対応したファイルを追加するとき

## 使い方

### 外観

#### コンポーネント要素：通常時（ファイル未選択）

通常時のコンポーネントは以下の要素で構成されます：

- **ファイル選択ボタン**（必須）
- **ドロップ領域拡大チェックボックス**
- **ドロップエリア**
- **項目ラベル**（必須）
- **要否ラベル**
- **サポートテキスト**
- **ファイル未選択メッセージ**（必須）

ドロップエリアは、ファイル選択ボタンの後ろのテキスト「または、このエリア内にドラッグ＆ドロップ」を伴います。ボタンとチェックボックス間は最低 56 CSS px の空きを確保します。

#### コンポーネント要素：ファイル選択時、エラー

ファイル選択時に「選択ファイルサマリー」が表示されます。エラー発生時は「エラーテキスト（ファイル選択ボタンに関するエラー時必須）」が伴います。選択ファイル一覧では各行に以下が表示されます：

- 解除ボタン
- ファイル名
- メタ情報

### ドラッグ＆ドロップ範囲拡大

チェックボックスでユーザーが範囲拡大を選択できます。有効時、ドラッグ＆ドロップの範囲はブラウザのビューポート領域の全体となります。ファイルドラッグ状態でメッセージが表示され、全体がドロップエリアになります。

## アクセシビリティ

### ドラッグ操作への依存禁止

ドラッグ操作に依存しないデザインが必須です。上肢障害、振動障害、チック症、あるいはマウスの操作自体に不慣れだったり力が入らない状況等、マウスによる項目のドラッグが難しい環境のユーザーのために、必ずファイル選択ボタンを配置する必要があります。

ドロップエリアのみでの実装は許可されておらず、ファイル選択ボタンを通じてドラッグなしのシングルポインタで完遂できるようにしてください。
