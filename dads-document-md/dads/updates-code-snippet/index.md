## 2024年12月9日

- **Selectコンポーネント**
  - コンポーネントスタイルから`min-width`を削除など、汎用性を高めるための改修を実施
  - サンプルのメニュー（`option`）がFirefoxでキーボード操作ができなかった問題を修正
- **Inputコンポーネント**
  - コンポーネントスタイルから`min-width`を削除
- **Textareaコンポーネント**
  - コンポーネントスタイルから`min-width`を削除
  - コンポーネントスタイルから`field-sizing: content`を削除し、入力内容によって高さが可変するテキストエリアのサンプルの提供のみに変更
- **Tableコンポーネント**
  - `Table`コンポーネントとそれに関連する`Th`、`Td`コンポーネントなどを削除し、HTML要素を使ったサンプルの提供のみに変更
- **その他の修正**
  - tailwind-theme-pluginのアップグレードに伴い、廃止予定のクラスを新しいものに置き換え（例：`leading-1-7` → `leading-170` など）
  - Storybookプレビュー画面のfont-familyにデザイントークンで定義されたTailwind CSSクラスを使用
